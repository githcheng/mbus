儋州市滨海新区供排水一体化PPP项目
发布时间：2016-06-15 17:24:59　　

华诚博远（北京）投资顾问有限公司关于儋州市滨海新区供排水一体化PPP项目的公告

华诚博远（北京）投资顾问有限公司受儋州市农业委员会委托，对儋州市滨海新区供排水一体化PPP项目 进行 竞争性磋商 ，现邀请国内合格的供应或制造商来参加密封投标。

1. 招标编号： HCBY-2016-008

2. 招标项目及范围：儋州市滨海新区供排水一体化PPP项目（1个包）。

包号：项目本身
本项目包括供水项目和污水处理项目，为新建项目。供水项目，即新建滨海新区白马井供水厂及其配套管网；污水处理项目，即新建滨海新区白马井污水处理厂及其配套管网。

3. 供应商资格要求

3.1、3.1.1、依法成立并有效存续的境内外企业法人；
3.1.2、具有良好的银行资信、财务状况以及相应的投融资、偿债能力。截至投标前（即不早于2015年12月31日），经审计的企业净资产不低于人民币5亿元整（或等值外币，下同）；
3.1.3、在中国大陆地区至少有1个投资或运营管理城镇供水项目；社会资本或其控股的下属公司（包含直接或间接控股的下属公司，下同）、社会资本的母公司（即最终对社会资本控股的公司，下同）、母公司直接或间接控股的公司业绩都可列为社会资本业绩。社会资本提供的业绩是其控股的下属公司及母公司、母公司直接或间接控股的公司的，需要提供社会资本与其控股的下属公司及母公司、母公司直接或间接控股的公司的关系证明（出资关系的工商查询函或经审计的公司年度报告、企业产权登记证明文件等）。
3.1.4、在中国大陆地区至少有1个投资或运营管理城镇污水处理项目；社会资本或其控股的下属公司、社会资本的母公司、母公司直接或间接控股的公司业绩都可列为社会资本业绩。社会资本提供的业绩是其控股的下属公司、母公司、母公司直接或间接控股的公司的，需要提供社会资本与其其控股的下属公司、母公司、母公司直接或间接控股的公司的关系证明（出资关系的工商查询函或经审计的公司年度报告、企业产权登记证明文件等）
3.1.5、关联方约束；
3.1.6、本项目不接受联合体投标。
3.1.7、法律、法规规定的其他条件。
3.2、投标时必须提交以上相关证明资料。
