出海2年，5亿用户，欧拉网络分发游戏7天攻陷9国！你的轻手游也可以去赚美元

KUMA
标签: 游戏欧拉网络C轮娱托邦

欧拉网络是一家面向海外的移动应用开发商，坐标上海。2014年Q4转型海外市场后，旗下两款主要产品： Hola桌面(hola launcher)和单手划划(omni swipe)等工具类产品，以及其他移动应用17个月以来总共累计近5亿用户。去年欧拉网络入围了Google应用商店前十开发商榜单，并同时被Google和Facebook以案例的形式收录至其官方网站。

创始人Brian对创业邦（微信搜索：ichuangyebang）说，欧拉其实是一个积累了6年的老团队。之前一直是默默无闻，但是长时间的技术和经验积累，帮助他们一年多的时间在海外取得了爆发式的用户增长。

2016年3月，欧拉网络（holaverse）获得了7000万美元的C轮投资，毅达资本领投，奇虎360和执一资本跟投。基于积累了大量用户，欧拉对于转型进军国际游戏业务市场抱有坚定的信心。依靠自建投资回报测算系统，建立一套完整的出海方案，帮助国内的游戏开发商出海。

为什么欧拉目前的成绩还不错？

创业邦（微信搜索：ichuangyebang）认为，欧拉网络在出海之后取得这样的成绩有三个原因：

原因一：团队积累6年，技术成熟

欧拉网络的团队已经磨合了很长时间。在经历了国内工具类产品积累期之后，两年前转型出海产品。依靠之前积累的技术经验做出了靠谱的爆款工具。

原因二：具备时间差和空间差的商业模式

不可否认的是，互联网虽然无国界，但是各个地区的发展存在时间差和空间差，如果合理利用好这个“不对称”，就能产生很好的商业模式。

Brian说，过去几年在中国互联网发生的事情，目前在印度和巴西等地区就有“红利”，而全球未来几十亿的新增网民很可能都是靠手机触网，前景广阔。

这种利用空间和时间差异，将成熟的商业模式移植到新兴市场的模式，他称之为“Copy China To the Globe”。商业是追求本质的，只要根据商业逻辑去判断，就可以拥有预见性。

原因三：对于海外产品规律的把控

海外的成功产品有自己的成功规律，包括流量选择、地区差异。而移动互联网产品之所以更适合全球化，是因为渠道更加统一了。

下载有苹果的App Store，安卓的Google Play，流量推广则有Goolge和Facebook，全球化的门槛变低。

流量分发游戏的可行性

之所以选择游戏作为公司升级的起点，是因为游戏和工具都是基于流量的。

目前国内的流量已经基本被巨头把控，渠道和市面上的手游面临较高的流量费用。在这种情况下，中小型游戏开发商竞争压力增大，反观国际市场仍大有可为，不如转身拥抱海外市场。

可是做海外是要交学费的。包括游戏的特殊性、前期策划、美术风格都需要向海外市场靠拢。如果是非成熟的团队，制作一款足够出海的游戏，起码要用8-12个月的时间。这样在时间上会有滞后性。

目前，由欧拉网络（Holaverse）代理发行，Tuyoo&BoleGame研发的博彩游戏Kingslots已跻身Google Play排行榜三甲,其中在用户消费能力最强、付费意愿最高排过美国第二。

欧拉游戏业务总经理庞磊对创业邦（微信搜索：ichuangyebang）说，欧拉自建的ROI测算系统能根据大数据的采集选定国家，精细化市场投放与流量经营，做到精准定位用户。

这次被选中投放的9个国家，由Tuyoo&BoleGame研发的游戏Kingslots都名列前茅，其中美国、澳大利亚、意大利、西班牙和加拿大冲进了Top New Casino榜单三甲，而新西兰、新加坡和德国跻身前六。

工具类流量变现的新方向

Brian说，欧拉网络在今年转型的契机，是基于公司对于互联网流量的思考。当然，并非所有的游戏都适合用工具类流量来变现。

Brian告诉创业邦（微信搜索：ichuangyebang），欧拉游戏业务今后主要代理的游戏需要具备两种特征：

1、轻度的单机游戏，或者轻中度的网游。

重度网游更适合立体式的多渠道推广。在这方面，国内也有很多深耕多年的出海发行商。

2、全球普适性较高的游戏。

类似跑酷、切水果等游戏能风靡全球，就是因为它们适合所有人群。

区域文化特征很重的游戏并不能单纯用工具流量来推广。

在后期的游戏选择上，需要更加专注品质的筛选，这样才可以在用户红利变弱（即用户数在平台的增量放缓）时继续发挥流量的作用。

同时，欧拉网络购买流量的方案，以及更精准的投资回报测算也是对今后的流量推广起到关键作用的技术。
