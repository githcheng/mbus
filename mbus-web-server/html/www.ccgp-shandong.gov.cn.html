济宁医学院附属医院济宁医学院附属医院第二手术室改造工程公开招标公告

( 招标公告的公告期限为5个工作日 )
一、采购项目名称：济宁医学院附属医院第二手术室改造工程
二、采购项目编号：PHZB2016-124
三、采购内容及分包情况：
包号货物名称供应商资格要求本包预算金额(万元)
1济宁医学院附属医院第二手术室改造工程 （1）具有独立法人和合法经营资格的投标人，能有效的履行合同，有良好的商业信誉和较强的经营实力；（2）同时具有建筑主管部门颁发的建筑装修装饰工程专业承包（二级及以上资质）、建筑机电安装工程专业承包（二级及以上资质）及建筑智能化工程专业承包（二级及以上资质）；（3）项目经理具有相关专业贰级及以上建造师资格，且在本公司注册，具备有效的安全生产考核合格证书（B证），无在建工程；（4）具有有效的安全生产许可证；（5）未处于被责令停业、投标资格被取消或者财产被接管，冻结和破产状态；（6）没有因骗取中标或者严重违约等问题，被有关部门暂停投标资格并在暂停期内等情形；（7）本项目不接受联合体投标。 190.000000 
四、获取招标文件
  1.时间：2016年6月15日8时30分至2016年6月21日17时0分（北京时间，法定节假日除外）
  2.地点：济南市历下区历山路历山名郡C5座西单元二楼大厅
  3.方式：本项目要求现场报名，报名时请携带营业执照副本、施工资质证书副本、建造师证书、安全生产考核合格证书（B）证、安全生产许可证、法人证或法人授权委托书及本人身份证，以上资料原件（施工资质证书可除外）及加盖公章的复印件一套。
  4.售价：300元
五、递交投标文件时间及地点
  1.时间：2016年7月8日8时30分至2016年7月8日9时30分（北京时间）
