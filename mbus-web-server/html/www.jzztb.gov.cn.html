荆州市青少年宫购买后勤社会化服务项目竞争性磋商公告
[点击次数:27530  更新时间:2015-12-30]

 

根据荆州市政府采购管理办公室[2015]3317号计划函要求，荆州市公共资源交易中心受荆州市青少年宫的委托，对荆州市青少年宫购买后勤社会化服务项目进行竞争性磋商，现邀请国内具有相应资格的供应商前来响应磋商。
1.项目编号：JZC-2015124357
2.竞争性磋商项目：荆州市青少年宫购买后勤社会化服务
3.本项目采购预算:42万元
特别说明：荆州市青少年宫购买后勤社会化服务成交人需完成收费经济指标：人民币37万元既租赁单位物业管理费，院内停车收费；荆州市青少年宫实际支付费用5万元人民币。
4.合格的磋商响应供应商：
（1）磋商响应供应商具备《中国人民共和国政府采购法》第二十二条规定的条件；
（2）磋商响应供应商必须具备住房和城乡建设部颁发的物业管理三级资质证书；
（3）本项目不接受联合体投标。
5.磋商保证金：磋商供应商必须提供由磋商小组认可，已交纳磋商保证金的有效证明。所有响应供应商应于磋商前向磋商组织人专户汇入￥4000.00元(大写：肆仟元整)的磋商保证金，并办妥所有财务手续。
开户行：中国建设银行荆州市新桥支行
开户名：荆州市公共资源交易中心保证金专户
帐 号：42001625208053001890
6.领取磋商文件时间及地点：2015年12月30日至2016年1月6日，上午8：30分至12：00分，下午14：30时至17：30分；荆州市公共资源交易中心（荆州市北京西路440号，荆州市政务服务中心六楼）。
7.磋商文件的获取：
获取本磋商文件时需携带以下文件原件（不接受公证件）及加盖响应人公章
的复印件一套（复印件留存荆州市公共资源交易中心磋商时核对）：
（1）企业法人营业执照、税务登记证、组织机构代码证；
（2）法人身份证或法人授权委托书及身份证；
（3）住房和城乡建设部颁发的物业管理三级资质证书。
8、项目说明会时间：磋商响应供应商在领取采购文件后，于2016年1月7日上午 10:00时到荆州市青少年宫集中进行项目说明。
项目说明会联系人：杨昌喜               联系电话：13872408501
9.递交磋商响应文件截止时间：2016年1月11日上午9：00前（北京时间）在磋商地点递交磋商组织人荆州市公共资源交易中心，逾期或不符合规定的磋商响应文件恕不接受。资格原件随身携带，在磋商前供磋商小组核实。
10.磋商时间：2016年1月11日上午9：00开始。
11.磋商地点：荆州市公共资源交易中心（荆州市北京西路440号，荆州市政务服务中心六楼）。
12.本磋商公告首发媒体：
   湖北政府采购网：www.ccgp-hubei.gov.cn
 荆州招投标信息网：www.jzztb.gov.cn
